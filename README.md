# Creative Counseling

This is a counseling website built for Creative Counseling, in Winston-Salem, NC.

This was project I built using Next.js and later integrated Sanity for a CMS.

## Live build is being hosted on Vercel as of (04/2026)

The company has since had a remodel of the website since It was completed, and now just serves as a portfolio item. I've disabled all contact links and removed all contact information.

You can view the site here [Creative Counseling](https://creative-counseling.vercel.app/)
